# Logokitty — Test Cases

## 1. Auth & Onboarding

### TC-1.1: Первый запуск — показать экран логина
- **Pre**: localStorage пуст
- **Step**: Загрузить `/`
- **Assert**: URL → `#/auth`, видна кнопка «Войти через Telegram» и «Начать без Telegram»

### TC-1.2: Создать ребёнка и котика
- **Pre**: На странице `#/auth`
- **Step**: 
  1. Нажать «Начать без Telegram»
  2. Ввести имя родителя: «Анна»
  3. Ввести имя ребёнка: «Миша», возраст: 3
  4. Ввести имя котика: «Барсик», цвет: gray
  5. Нажать «Начать заниматься!»
- **Assert**:
  - URL → `#/feed`
  - localStorage.lk_user.name === «Анна»
  - localStorage.lk_children[0].name === «Миша», age === 3, level === 1, foodBalance === 50
  - localStorage.lk_cats[0].name === «Барсик», color === «gray», equippedItems === []

### TC-1.3: Повторный запуск — редирект на Feed без онбординга
- **Pre**: localStorage содержит User + Child
- **Step**: Перезагрузить страницу
- **Assert**: URL → `#/feed`, без показа экрана логина

### TC-1.4: Валидация полей при создании
- **Pre**: На странице создания
- **Step**: Нажать «Начать заниматься!» с пустым именем ребёнка
- **Assert**: Ничего не происходит (кнопка не активна или имя не пустое)

### TC-1.5: Выбор цвета котика
- **Pre**: На странице создания котика
- **Step**: Нажать на серый кружок
- **Assert**: Состояние цвета котика — gray, отображается 🔘

## 2. Feed (Лента)

### TC-2.1: Отображение ленты после создания
- **Pre**: Создан ребёнок
- **Step**: Открыть `#/feed`
- **Assert**:

  - Видна карточка ежедневного бонуса (+50 🐟)

  - Виден раздел «Последние достижения» с 3 заглушками

### TC-2.2: Забрать ежедневный бонус
- **Pre**: Feed открыт, бонус не забран
- **Step**: Нажать «Забрать»
- **Assert**:

### TC-2.3: Ежедневный бонус нельзя забрать дважды

- **Step**: Перезагрузить страницу

### TC-2.4: Отображение второго ребёнка
- **Pre**: В localStorage два Child с разными id
- **Step**: Перезагрузить, открыть `#/feed`
- **Assert**: В ленте отображается первый ребёнок, раздел «Дети» показывает обоих

### TC-2.5: Переключение между детьми
- **Pre**: 2+ ребёнка
- **Step**: Нажать на второго ребёнка в разделе «Дети»
- **Assert**: activeChildId меняется, cat загружается для выбранного ребёнка

## 4. Games (Игры)

### TC-4.1: PhonemicFind — правильный ответ
- **Pre**: Открыта игра «Найди звук» (id=g1)
- **Step**: Нажать на карточку с правильным звуком (hasSound=true)
- **Assert**:
  - Карточка подсвечивается зелёным (#e8f5e9)
  - Через 800ms переходит к следующему раунду
  - После 3 успешных раундов: игра завершена, +30 EXP, +5 🐟

### TC-4.2: PhonemicFind — неправильный ответ
- **Pre**: Открыта игра «Найди звук»
- **Step**: Нажать на карточку без нужного звука (hasSound=false)
- **Assert**:
  - Карточка подсвечивается красным (#ffebee)
  - Через 800ms показывается «Попробуй ещё!» (lose feedback)
  - Через 1000ms игра возвращается в исходное состояние

### TC-4.3: MemoryMatch — найти пары
- **Pre**: Открыта игра «Найди пару» (id=g6)
- **Step**: Последовательно открыть 2 одинаковые карточки
- **Assert**: Карточки остаются открытыми (matched=true), счётчик ходов = 1

### TC-4.4: MemoryMatch — несовпадающие карточки
- **Pre**: Открыта игра «Найди пару»
- **Step**: Открыть 2 разные карточки
- **Assert**: Через 1000ms карточки закрываются обратно, счётчик ходов = 1

### TC-4.5: MemoryMatch — все пары найдены → game complete
- **Pre**: Игра MemoryMatch (4 пары = 8 карточек)
- **Step**: Найти все 4 пары
- **Assert**: Через 500ms onComplete(true), +25 EXP, +5 🐟

### TC-4.6: Level-up после игры
- **Pre**: Ребёнок level=1, exp=90 (нужно 100 для lvl 2)
- **Step**: Пройти игру с expReward=30
- **Assert**:
  - EXP: 90+30=120 → lvl up: 120-100=20 → level=2, exp=20
  - Показан LevelUpModal «Уровень 2!»
  - Нажатие «Ура!» → закрывает модалку

### TC-4.7: Уровень кратен 5 — начисление золота
- **Pre**: Ребёнок level=4, exp=95
- **Step**: Пройти игру с expReward=30
- **Assert**: 95+30=125 → 125-400=25 → level=5, exp=25, gold += 10

### TC-4.8: Кнопка «Назад» в игре
- **Pre**: Игра открыта
- **Step**: Нажать «← Назад»
- **Assert**: URL → `#/section/{game.section}`

### TC-4.9: Игра с unlockLevel — недоступна
- **Pre**: Ребёнок level=1, игра «Подуй на шарик» требует level=2
- **Step**: Открыть раздел «Запуск речи»
- **Assert**: Игра «Подуй на шарик» не отображается в списке

## 10. Edge Cases

### TC-10.1: Очень длинные имена
- **Pre**: Создание ребёнка
- **Step**: Ввести имя ребёнка из 200 символов
- **Assert**: Имя отображается, не ломает вёрстку, localStorage сохраняет полное имя

### TC-10.2: Минимальный возраст
- **Pre**: Создание ребёнка
- **Step**: Установить возраст 2
- **Assert**: Ребёнок создан, age=2, все фильтры игр по возрасту работают

### TC-10.3: Максимальный уровень
- **Pre**: Ребёнок level=49, exp=4900
- **Step**: Добавить 200 EXP (игра + скармливание)
- **Assert**: level=50, exp=0 (или остаток), isStageComplete=true

### TC-10.4: Отрицательный баланс
- **Pre**: foodBalance=10
- **Step**: Попытаться купить товар за 100
- **Assert**: Кнопка disabled, баланс не меняется

### TC-10.5: Множественное нажатие в игре
- **Pre**: Игра PhonemicFind
- **Step**: Быстро нажать 3 карточки подряд
- **Assert**: Обрабатывается только первое нажатие (sel guard), остальные игнорируются (disabled)
